first_site
==========

## Workflow

```sh
# cloning repo from github to your laptop
git clone https://github.com/hannaht1991/first_site.git
# select the gh-pages branch, the one that GitHub looks for your website in
git checkout gh-pages
# make some changes... e.g. add styling
# mark changes for commiting
git add .
# commit your changes and describe what you've done
git commit -m "added styling"
# share those changes with GitHub, which can then update your site
git push
# view website
# open http://hannaht1991.github.io/first_site/
```
